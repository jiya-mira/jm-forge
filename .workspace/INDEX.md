# Workspace Index

Runtime workflow artifacts root.

## Managed Files
- `INDEX.md`: top-level routing entry for `.workspace/`.

## Managed Directories
- `tasks/`: all task artifacts (`discuss/plan/execute/summary` by task).
- `project-map/`: project context map (entry -> nodes -> relationships).
- `resource-map/`: project-local resource inventory.
- `exp-map/`: project-level experience records.

## Navigation
1. Open `tasks/INDEX.md` to locate a task directory.
2. Open `project-map/INDEX.md` for codebase context cascade.
3. Open `resource-map/INDEX.md` for resource data ownership.
4. Open `exp-map/INDEX.md` for reusable experience catalog.

## Update Rule
Update this file when top-level directories or navigation entry paths change.
