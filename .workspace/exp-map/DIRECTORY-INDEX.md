# EXP Directory Index

Directory routing guide for `.workspace/exp-map/`.

## Managed Files
- `DIRECTORY-INDEX.md`: folder guide and navigation contract.
- `INDEX.md`: tabular EXP catalog.

## Managed Directories
- `entries/`: EXP detail files (`<id>.md`).

## Navigation
1. Start with `INDEX.md` to locate target EXP.
2. Open `entries/<id>.md` for full detail.

## Update Rule
Update when `entries/` structure or EXP navigation contract changes.
